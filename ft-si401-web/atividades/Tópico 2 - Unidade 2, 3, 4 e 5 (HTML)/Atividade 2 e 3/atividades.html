Atividade 1 - Introdução

  1. Apresente (e explique brevemente) todos os códigos de status numéricos, com as respectivas descrições, que podem ser retornados em uma resposta HTTP/1.1.

  2. O Chrome DevTools (ou Ferramentas do Desenvolvedor é um conjunto de ferramentas de autoria e depuração nativamente incorporado ao navegador Chrome Consulte a documentação do Chrome DevTools link abaixo) e apresente uma descrição de todos os painéis disponíveis pela ferramenta

    https://developers.google.com/web/chrome/devtools

  3. Com o auxílio das Chrome DevTools acesse o endereço http://www.ft.unicamp.br e responda:

    • Quantas requisições são feitas ao servidor, até que a página seja exibida na íntegra?
    • Todas as requisições retornam status 200 ( Caso não, indique qual requisição retorna um código diferente, qual é o código em questão e qual o seu significado.
    • Apresente todos os campos dos cabeçalhos (de requisição e resposta que são trocados na requisição que retorna o document principal da página.
    • Há alguma mensagem de erro ou de warning exibida no console. Se sim, quais?

Atividade 2 - HTML

  1. Crie um conjunto de documentos HTML que represente uma versão preliminar de um site para a disciplina SI401.

    • Estes documentos devem usar o que foi visto até o momento no curso (Unidades 2 e 3)
    • Busque representar no site todos os principais aspectos da disciplina: ementa, objetivos, tópicos que serão tratados, referências, critérios de avaliação, etc.
    • Inclua imagens, referências a materiais externos e o que mais achar necessário.
    • Siga o formato HTML5 e valide todos os documentos gerados no site do W3C.
